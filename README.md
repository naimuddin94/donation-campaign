## Run Locally

Clone the project

```bash
  git clone https://github.com/programming-hero-web-course-4/b8a8-donation-campaign-naimuddin94.git
```

Go to the project directory

```bash
  cd b8a8-donation-campaign-naimuddin94

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
